IFTTT

IFTTT is a service that creates chains of conditional statements, called applets, that can be triggered by the events occurring within various web services. These applets connect multiple applications in order to run automated tasks.

To connect The Things Stack to IFTTT, Node-RED will also be used as a part of the setup.

Prerequisites

  1. A user account on IFTTT.

  2. A running instance of Node-RED.

Detailed instructions on using IFTTT with The Things Stack are in the topics below.
