Fine-tuning MAC Settings for End Devices

MAC settings on The Things Stack are configurable per end device. See the MAC settings guide for instructions.

Warning:
Devices imported from The Things Network Stack V2 are configured with an Rx1Delay of 1 second, by default. In The Things Stack we recommend using an Rx1Delay of 5 seconds to accommodate for high latency backhauls and/or peering with Packet Broker. Devices added to The Things Stack Cloud use an Rx1Delay of 5 seconds by default.
