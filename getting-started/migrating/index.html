Migrating to The Things Stack

This guide documents the process of migrating end devices and gateways to The Things Stack.

End devices from V2 or any other LoRaWAN network (like Chirpstack) can be migrated to The Things Stack. Migrating devices can be done in two ways - manually via The Things Stack Console or by using ttn-lw-migrate tool, depending on how many devices and which network you are migrating from.

Note:
In general, end devices from any LoRaWAN network can be imported to The Things Stack by using a properly formatted JSON file.

In this section, we first consider migrating end devices, then migrating gateways.
