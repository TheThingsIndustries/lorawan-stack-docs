Host The Things Stack

The Things Stack is a highly flexible software product that is available in a variety of deployment models and caters to various user requirements. Check which model applies to your use case.

The Things Industries offers The Things Stack as a fully-managed, SLA-backed cloud subscription called The Things Stack Cloud. The complexity of managing a LoRaWANÂ® network server in production is handled by us, while you can focus on delivering value to your customers.

Users can easily get started with The Things Stack Cloud by signing up for free discovery tier to get started and test the capabilities of The Things Stack. Once you are ready to scale your LoRaWAN fleet, you can seamlessly update the subscription to reflect the changes. For most users, The Things Stack Cloud is the recommended option.

For users with operational experience and for users who require that the LoRaWAN data is within their own infrastructural domain, The Things Stack is packaged in various flavours for users to install on their own hardware/cloud.

The Things Stack is available for production installations for the following cases

  • Amazon Cloud Services templates
  • Kubernetes templates

For users who want to install The Things Stack on their local machines for testing or want to contribute to The Things Stack open source repository, The Things Stack is also available to be installed via Docker.

Special Deployments

The Things Join Server is a stand-alone LoRaWAN Join Server. The Things Join Server can be used by device makers to provision end devices to be used by all distributions of The Things Stack as well as any other standards compliant LoRaWAN Network Servers.

AWS

The Things Stack is currently available as two flavours for self-hosting on Amazon Web Services (AWS). AWS Marketplace AMI An AWS CloudFormation stack that can be purchased at the AWS Marketplace. AWS ECS A set of CloudFormation templates that can be deployed to build a highly available cluster for large-scale deployments. In addition, The Things Stack natively supports interaction with AWS IoT. Check the AWS IoT guide to learn to integrate The Things Stack with the AWS IoT services. Note: For getting started, we recommend the AWS Marketplace AMI option.
