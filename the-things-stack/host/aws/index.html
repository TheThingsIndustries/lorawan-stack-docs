AWS

The Things Stack is currently available as two flavours for self-hosting on Amazon Web Services (AWS).

  1. AWS Marketplace AMI
  • An AWS CloudFormation stack that can be purchased at the AWS Marketplace.
  1. AWS ECS
  • A set of CloudFormation templates that can be deployed to build a highly available cluster for large-scale deployments.

In addition, The Things Stack natively supports interaction with AWS IoT. Check the AWS IoT guide to learn to integrate The Things Stack with the AWS IoT services.

Note:
For getting started, we recommend the AWS Marketplace AMI option.
