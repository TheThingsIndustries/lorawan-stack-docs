Connect Milesight UG85 with Lora Basics™ Station

This section contains instructions for connecting the Milesight UG85 LoRaWAN® gateway to The Things Stack using LoRa Basics™ Station.

In the Packet Forwarder menu and General tab, click the little + button to create a new server.

In the server configuration options, check the Enable box.

Choose Basic Station as the Type.

Follow the instructions for Connecting LNS. If using CUPS, following the instructions for Connecting CUPS.

Click Save to continue.

If your configuration was successful, your gateway will connect to The Things Stack after a couple of seconds.
