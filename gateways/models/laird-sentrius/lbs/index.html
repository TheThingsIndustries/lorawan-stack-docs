Connect Laird Sentrius™ RG1xx with LoRa Basics™ Station

This section contains instructions for connecting to The Things Stack using LoRa Basics™ Station.

To set up LoRa packet forwarding on the gateway, follow these steps.

Click the LoRa tab in the main menu.

Click on the Forwarder tab.

Choose Semtech Basic Station as the Mode.

CUPS

To connect the gateway via CUPS protocol, follow the instructions for Connecting CUPS.

Fill the CUPS Server in the Server Configuration with the CUPS Server Address.

Click on the Choose File button to upload the Server Certificate File and key File with the CUPS Server Certificate and CUPS Key File respectively.

Click on the Upload Certificates button.

Click Update. If your configuration was successful, your gateway will connect to The Things Stack after a couple of seconds.

LNS

To connect the gateway via LNS protocol, follow the instructions for Connecting LNS.

Fill the LNS Server in the Server Configuration with the LNS Server Address.

Click on the Choose File button to upload the Server Certificate File and key File with the LNS Server Certificate and LNS Key File respectively.

Click on the Upload Certificates button.

Click Update. If your configuration was successful, your gateway will connect to The Things Stack after a couple of seconds.
