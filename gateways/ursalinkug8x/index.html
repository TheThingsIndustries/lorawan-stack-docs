Ursalink UG8X IoT LoRaWAN Gateway

The Ursalink UG8X IoT LoRaWAN Gateway is an 8 channel (16 channel optional) configurable, scalable gateway for industrial IoT applications.

This page contains information about connecting the Ursalink UG8X IoT LoRaWAN Gateway to The Things Stack

The technical specifications can be found in Ursalink’s official documentation. The Ursalink UG8X IoT LoRaWAN Gateway supports two ways to connect with The Things Stack, using either the Semtech Packet Forwarder or LoRa Basics™ Station.

Requirements

  1. User account on The Things Stack with rights to create gateways.
  2. Ursalink UG8X LoRaWAN Gateway connected to the internet via ethernet or cellular backhaul.
  3. CA certificate for LoRa Basics™ Station (if using LoRa Basics™ Station).

Registration

Create a gateway by following the instructions for the Console or the CLI.

The EUI of the gateway can be found on the configuration web page of the gateway. See the next section for instructions to access the configuration page.

Configuration via Browser

Find the IP address of the gateway. The default IP for the Ursalink UG8X LoRaWAN Gateway is 192.168.23.150.

Connect your machine to the same local network as that of the gateway, and enter the IP address in your web browser. The default username is admin and the default password is password. See Ursalink’s official documentation for more information.

Disable Default Server

In the left menu, choose Packet Forwarder. Select the General tab.

Click the pencil icon next to the default server, and uncheck the Enabled button to disable the default server.

Click Save to continue.

Connect to The Things Stack

After completing basic configuration, follow the instructions for connecting using LBS or the UDP Packet Forwarder.
