Milesight UG6X LoRaWAN Gateway

The Milesight UG6X is a series of 2 robust 8-channel LoRaWAN gateways for industrial IoT applications. This page contains information about connecting the Milesight IoT UG6x LoRaWAN gateway to The Things Stack.

The technical specifications can be found in Milesight’s official UG65 and UG67 pages.

The Milesight UG6X LoRaWAN gateways support two ways of connecting The Things Stack, using the Semtech UDP Packet Forwarder or LoRa Basics™ Station.

Requirements

  1. User account on The Things Stack with rights to create gateways.
  2. Milesight IoT UG6X LoRaWAN gateway connected to the Internet via Ethernet, Wi-Fi or cellular backhaul.
  3. CA certificate for LoRa Basics™ Station (if using LoRa Basics™ Station).

Registration

Add a gateway by following the instructions for the Console or the CLI.

The Gateway EUI of the gateway can be found on the gateway’s configuration web page. See Configuration via Browser section below for instructions to access the configuration page.

Configuration via Browser

Find the gateway’s IP address. The default IP address for the Milesight UG6X LoRaWAN gateway is 192.168.23.150.

Connect your machine to the same local network as that of the gateway, and enter the gateway’s IP address in your web browser. YOu will be presented with a login screen. The default username is admin and the default password is password.

See Milesight’s official documentation for more information.

Disable Default Server

In the left menu, choose Packet Forwarder. Select the General tab.

Click the pencil icon next to the localhost, and uncheck the Enable button to disable the Embedded NS.

Click Save to continue.

Connect to The Things Stack

After completing the basic configuration, follow the instructions to Connect Milesight UG6X with LoRa Basics™ Station or to Connect Milesight UG6X with Semtech UDP Packet Forwarder.
