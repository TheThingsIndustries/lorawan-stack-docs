LoRaWAN Specification and Regional Parameters

The Things Stack supports versions 1.0.x and 1.1.x of the LoRaWAN specifications.

This includes all of the following LoRaWAN specification versions:

  • LoRaWAN 1.0
  • LoRaWAN 1.0.1
  • LoRaWAN 1.0.2
  • LoRaWAN 1.0.3
  • LoRaWAN 1.0.4
  • LoRaWAN 1.1

The LoRaWAN specification versions are available from the LoRa Alliance resource hub.

The Things Stack supports the following LoRaWAN Regional Parameter specification versions:

  • LoRaWAN Regional Parameters 1.0
  • LoRaWAN Regional Parameters 1.0.1
  • LoRaWAN Regional Parameters 1.0.2 Rev A
  • LoRaWAN Regional Parameters 1.0.2 Rev B
  • LoRaWAN Regional Parameters 1.0.3 Rev A
  • LoRaWAN Regional Parameters 1.1 Rev A
  • LoRaWAN Regional Parameters 1.1 Rev B

The LoRaWAN Regional Parameters specification versions are available from the LoRa Alliance resource hub.

In addition, The Things Stack supports Class A, B, C, and multicast group devices. See the Class B, Class C, and Multicast references for special instructions when using these device classes.
