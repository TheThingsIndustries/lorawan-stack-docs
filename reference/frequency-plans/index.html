Frequency Plans

This reference contains Frequency Plans that are officially supported by The Things Stack. This list is populated from the lorawan-frequency-plans Github repository.

Deployments may also have custom frequency plans configured.

Europe 863-870 MHz (SF12 for RX2)
Default frequency plan for Europe
Base Frequency: 868 MHz
ID: EU_863_870
Europe 863-870 MHz (SF9 for RX2 - recommended)
TTN Community Network frequency plan for Europe, using SF9 for RX2
Base Frequency: 868 MHz
ID: EU_863_870_TTN
United States 902-928 MHz, FSB 1
Default frequency plan for the United States and Canada, using sub-band 1
Base Frequency: 915 MHz
ID: US_902_928_FSB_1
United States 902-928 MHz, FSB 2 (used by TTN)
TTN Community Network frequency plan for the United States and Canada, using sub-band 2
Base Frequency: 915 MHz
ID: US_902_928_FSB_2
Australia 915-928 MHz, FSB 1
Default frequency plan for Australia, using sub-band 1
Base Frequency: 915 MHz
ID: AU_915_928_FSB_1
Australia 915-928 MHz, FSB 2 (used by TTN)
TTN Community Network frequency plan for Australia, using sub-band 2
Base Frequency: 915 MHz
ID: AU_915_928_FSB_2
Australia 915-928 MHz, FSB 6
Frequency plan for Australia, using sub-band 6, which overlaps with Asia 923-925 MHz
Base Frequency: 915 MHz
ID: AU_915_928_FSB_6
China 470-510 MHz, FSB 11
TTN Community Network frequency plan for China, using sub-band 11
Base Frequency: 470 MHz
ID: CN_470_510_FSB_11
Asia 920-923 MHz
TTN Community Network frequency plan for Asian countries, using frequencies ≤ 923 MHz
Base Frequency: 915 MHz
ID: AS_920_923
Asia 920-923 MHz with LBT
TTN Community Network frequency plan for Asian countries, using frequencies ≤ 923 MHz with listen-before-talk
Base Frequency: 915 MHz
ID: AS_920_923_LBT
Asia 923 MHz with only default channels
TTN Community Network frequency plan for Asian countries, using default frequencies of the AS923 band. This is added only for compatibility. Preferrably use the 920-923 or 923-925 variants
Base Frequency: 915 MHz
ID: AS_923
Asia 923-925 MHz
TTN Community Network frequency plan for Asian countries, using frequencies ≥ 923 MHz
Base Frequency: 915 MHz
ID: AS_923_925
Asia 923-925 MHz with LBT
TTN Community Network frequency plan for Asian countries, using frequencies ≥ 923 MHz with listen-before-talk
Base Frequency: 915 MHz
ID: AS_923_925_LBT
Asia 923-925 MHz (used by TTN Australia)
TTN Community Network frequency plan for Asia 923-925 MHz in Australia
Base Frequency: 915 MHz
ID: AS_923_925_TTN_AU
South Korea 920-923 MHz
TTN Community Network frequency plan for South Korea
Base Frequency: 915 MHz
ID: KR_920_923_TTN
India 865-867 MHz
Default frequency plan for India
Base Frequency: 868 MHz
ID: IN_865_867
Russia 864-870 MHz
TTN Community Network frequency plan for Russia
Base Frequency: 868 MHz
ID: RU_864_870_TTN
LoRa 2.4 GHz with 3 channels draft 2
LoRa 2.4 GHz plan with 3 channels for global use draft 2
Base Frequency: 2450 MHz
ID: ISM_2400_3CH_DRAFT2
