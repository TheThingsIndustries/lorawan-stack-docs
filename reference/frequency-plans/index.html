Frequency Plans

This reference contains Frequency Plans that are officially supported by The Things Stack. This list is populated from the lorawan-frequency-plans Github repository.

Deployments may also have custom frequency plans configured.

Europe 863-870 MHz (SF12 for RX2)

ID Base Frequency Description
EU_863_870 868 Default frequency plan for Europe

Europe 863-870 MHz (SF9 for RX2 - recommended)

ID Base Frequency Description
EU_863_870_TTN 868 TTN Community Network frequency plan for Europe, using SF9 for RX2

Europe 863-870 MHz, 6 channels for roaming (Draft)

ID Base Frequency Description
EU_863_870_ROAMING_DRAFT 868 European 6 channel plan used by major operators to support LoRaWAN Passive Roaming

Europe 433 MHz (ITU region 1)

ID Base Frequency Description
EU_433 433 Default frequency plan for worldwide 433MHz
