Command ttn-lw-cli applications packages associations list

Description

List application package associations

Aliases

ls

Usage

ttn-lw-cli applications packages associations list [application-id] [device-id] [flags]

Flags for ttn-lw-cli applications packages associations list

--all

type: bool

default: false

select all application package association fields

--application-id

type: string

--data

type: bool

default: false

select the data field and all allowed sub-fields

--dev-eui

type: string

(hex)

--device-id

type: string

--join-eui

type: string

(hex)

--limit

type: uint32

default: 50

maximum number of results to get

--package-name

type: bool

default: false

select the package_name field

--page

type: uint32

default: 1

results page number

This command inherits flags from ttn-lw-cli applications packages associations
